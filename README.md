## About the Project

**Animated Cards Opening** is a premium interactive UI built with HTML, CSS, JavaScript, and GSAP. It features smooth full-screen transitions, dynamic card reordering, elegant typography, and responsive animated details that create a cinematic browsing experience.

The layout is designed to feel modern, polished, and engaging, with clean motion effects and a strong visual hierarchy.

## Features

- Full-screen animated hero card transitions.
- Smooth GSAP-powered motion.
- Dynamic content updates on navigation.
- Elegant pagination and progress indicator.
- Responsive layout behavior.
- Premium dark theme with bold typography.
- Lightweight, fast, and visually immersive UI.
